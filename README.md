# NIGHTGRID

Skupinový projekt s Cyberpunk 2077 tématikou

## Technologie

**Client:** React + Vite, shadcn/ui, TailwindCSS

**Server:** Node, Express, MongoDB

## Instalace a spuštění

Client

```bash
  cd client
  npm install
  npm run dev
```
Server

```bash
  cd server
  npm install
  npm start
```
